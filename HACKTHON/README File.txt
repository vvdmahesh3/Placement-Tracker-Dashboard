# Placement Tracker Dashboard

## Overview
The Placement Tracker Dashboard is a web application designed to help students track their job applications, interviews, and skills. It provides a centralized platform for managing placement-related activities.

## Features
- **Application Tracker**: Add, edit, and delete job applications.
- **Interview Scheduler**: Schedule and manage interviews.
- **Skill Tracker**: Track and update your skills.
- **Advanced Features**: Notifications, analytics, and more.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript, Firebase Hosting
- **Backend**: Firebase Firestore, Firebase Authentication, Firebase Functions
- **Other Tools**: Firebase Analytics, Firebase Storage

## Why This Project is Good for Students
- **Centralized Platform**: Students can manage all placement-related activities in one place.
- **Real-Time Updates**: Firebase Firestore provides real-time updates for applications and interviews.
- **Skill Development**: Students can track their skills and identify areas for improvement.

## How to Use
1. Clone the repository.
2. Install dependencies using `npm install`.
3. Deploy the app using `firebase deploy`.
4. Open the provided Hosting URL in your browser.
